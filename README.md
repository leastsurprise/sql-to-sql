# MeshSafe New Zealand

**A LoRa mesh resilience concept for public safety in New Zealand.**

MeshSafe is an open, non-commercial concept for using low-cost
[LoRa](https://en.wikipedia.org/wiki/LoRa) mesh radios — running the open-source
[Meshtastic](https://meshtastic.org) firmware — to **reduce natural-hazard losses
and keep communities connected when the mobile network fails.**

When power and cell towers go down in a flood, cyclone, earthquake or slip —
as they did across Hawke's Bay and Tairāwhiti during Cyclone Gabrielle in 2023 —
households and responders lose contact exactly when they need it most. A mesh of
cheap, battery- and solar-powered radios keeps short text messages, location
beacons and emergency alerts flowing with **no tower, no SIM and no subscription.**

This repository contains a self-contained HTML slide deck presenting the idea,
aimed at New Zealand's natural-hazard agencies, councils and CDEM groups — for
example the [Natural Hazards Commission Toka Tū Ake](https://www.naturalhazards.govt.nz).

## Status

This is **an unfunded idea offered to the public interest** — a concept for
discussion, not a finished proposal, and not engineering, legal or
emergency-management advice. Any real deployment should be designed and tested
with the relevant technical experts and emergency-management authorities.
"MeshSafe" is used here as a working project name only.
